# AttendNow Web App

AttendNow is a comprehensive smart web-based attendance application designed to streamline the process of marking and tracking student attendance using QR codes and modern web technologies.

## Features

### Student Module
- **Dashboard**: Central hub for managing attendance and profile.
- **Mark Attendance**: Scan QR codes generated by teachers to mark attendance instantly.
- **Secure Login**: Supports WebAuthn for passwordless login using passkeys (fingerprint, face scan).
- **Performance**: View attendance statistics and progress bars for each subject.
- **Requests**: Contact admin for profile corrections and track request status.

### Teacher Module
- **Session Management**: Create attendance sessions for specific subjects, departments, and years.
- **QR Code Generation**: Generate dynamic QR codes with configurable validity and time limits.
- **Reports**: View attendance lists and export data to CSV.
- **Subject Management**: View assigned subjects and schedule.

### Admin Module
- **User Management**: Manage students, teachers, and their credentials.
- **Academic Structure**: Configure departments, subjects, semesters.
- **Promotions**: Automated tools to promote students to the next academic year.
- **Dashboard**: Real-time overview of registered students and daily sessions.

## Tech Stack

- **Backend**: Python (Flask)
- **Frontend**: HTML5, CSS3 (Bootstrap 5), JavaScript
- **QR Scanning**: `html5-qrcode` library
- **Authentication**: Standard session-based auth & WebAuthn integration

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ta-gthb/Attend_Now.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access the app at local server http://localhost:5000 or at public URL https://attendnow.onrender.com/.
